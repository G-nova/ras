R.A.S. : Responsive Accessible Slideshow

Comment installer RAS ?

<script src="ras.min.js" type="text/javascript"></script>
<script type="text/javascript">
  $.fn.ready(function(){
    $('#slideshow').ras();
  });
</script>

Options de ras

$('#slideshow').ras({
  duration:400, //durée du slide
  textPrev:'prev', //texte pour le bouton prev
  textNext:'next' //texte pour le bouton next
});

Ras Autoslide

<script src="ras-auto.min.js" type="text/javascript"></script>
<script type="text/javascript">
  $.fn.ready(function(){
    $('#slideshow').ras().autoslide();
  });
</script>

Options d'autoslide

$('#slideshow').ras().autoslide({
  time:2000, //durée entre chaque slide
  textPlay:'play', //texte pour le bouton play
  textStop:'stop' //texte pour le bouton stop
});

Ras Carousel

<script src="ras-auto.min.js" type="text/javascript"></script>
<script type="text/javascript">
  $.fn.ready(function(){
    $('#slideshow').ras().carousel();
  });
</script>

Options de carousel

$('#slideshow').ras().carousel({
  number:3, //groupe par n éléments (ici 3)
  oneVisible:'play', //slide 1 par 1 et laisse 1 de visible pour le responsive
});
